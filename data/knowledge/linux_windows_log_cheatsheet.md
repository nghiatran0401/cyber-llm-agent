# Linux and Windows Log Cheatsheet

## Windows Event IDs
- 4624 successful logon
- 4625 failed logon
- 4688 process creation
- 4720 user created
- 4740 account lockout

## Linux Signals
- Repeated SSH failures in auth logs
- Unusual sudo usage
- Unexpected cron/startup persistence changes
