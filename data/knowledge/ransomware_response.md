# Ransomware Response Guide

## Indicators
- Mass file encryption behavior
- Ransom notes in directories
- Security tooling tampering

## First Actions
- Isolate affected hosts
- Disable compromised accounts
- Preserve forensic artifacts
- Notify IR leadership/legal as needed

## Recovery
- Restore from known-good backups
- Rotate credentials and monitor for reinfection

## Source/Reference

- CISA Stop Ransomware
- NIST SP 800-61
