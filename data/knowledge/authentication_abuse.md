# Authentication Abuse Detection and Response

## Attack Types
- Brute force
- Password spray
- Credential stuffing

## Detection Signals
- Failure spikes, impossible travel, or fail-then-success patterns.

## Response
- Block abusive sources.
- Enforce MFA and reset credentials.
- Revoke active sessions and inspect privileged changes.

## Source/Reference

- MITRE ATT&CK T1110
- NIST identity guidance
