Large Language Model (LLM)-Powered Autonomous Agents and Multi-Agent Systems
Deadline: 11:59 PM, 02/11/2025 (End of Week 12)
Weight: 50% of final grade
Group Assignment: 3–5 students per group

Eligibility (Option D)
To undertake Option D, you must apply to the unit convenor by demonstrating your suitability for this research-oriented project. Requirements include:

A current academic average of at least Distinction (≥ 70%)

Aiming to achieve a Distinction/High Distinction in Intelligent Systems

Strong Python programming skills

Project Summary
This project involves working in a team led by teaching staff to implement and demonstrate autonomous agents powered by large language models (LLMs) such as GPT, LLaMA, or Gemini.

Intro resource: https://botpress.com/blog/llm-agents
(Students are expected to conduct their own research to find relevant learning materials.)

Project Goals
There are two main objectives:

(G1) Develop and deploy an LLM-powered autonomous agent

(G2) Develop and deploy a multi-agent system consisting of interacting LLM-powered agents

While both goals are preferred, teams may focus on achieving (G1) and making as much progress as possible toward (G2) if challenges arise.

Application Focus
Your system should focus on one of the following:

Business Improvement Agent

Analyzes company or organization reviews

Summarizes customer sentiment

Identifies areas for improvement

Cybersecurity Support Agent

Analyzes system logs to detect and diagnose issues

Identifies and predicts cyber attacks

Uses Cyber Threat Intelligence (CTI) for proactive defense and incident response

Project Requirements
Implementation must be in Python (not suitable for beginners)

Source code must be maintained using Git (GitHub, GitLab, Bitbucket, etc.) with read-only access for supervisors

Documentation must be maintained on OneDrive with supervisor access

Development requirements:

Learn LLMs, agentic AI, and multi-agent systems

Implement a basic LLM agent (locally and via cloud deployment)

Develop agents aligned with your team’s agreed mission

Build a multi-agent system with collaborative agents

Implement a user interface (UI)

Demonstrate a working prototype (preferably hosted for browser access)

Evaluate system performance using benchmarks/datasets
